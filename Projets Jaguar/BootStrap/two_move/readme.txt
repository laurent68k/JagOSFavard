	This simple program display the jaguar logo and French, German Flags int the screen,
in Video mode is CRY. And more, the Jaguar logo move !!

In this example i make an object list with two bitmap objects and one stop object.


* Boot.s is the main code where is initialized the jaguar

* Objcpu.inc contains the Object processor routines


CRY images:
-----------

* jaglogo.tga (Jaguar Logo):	160x58 pixels
* flags.tga (Fr & D flags):		64x64 pixels
