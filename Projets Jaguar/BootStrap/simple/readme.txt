	This simple program display the jaguar logo to the screen in Video mode is CRY
In this example i make a small object list with one bitmap object and one stop object.


* Boot.s is the main code where is initialized the jaguar

* Objcpu.inc contains the Object processor routines


CRY images:
-----------

* jaglogo.tga (Jaguar Logo):	160x58 pixels
* flags.tga (Fr & D flags):		64x64 pixels
* colours.tga (Colours Mire):	64x64 pixels
