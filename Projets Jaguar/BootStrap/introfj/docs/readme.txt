INTRODUCTION:
-------------

	This is a complete startup code to replace the orginal ATARI Jaguar's ROM.
This one must act nearly like the original did.

FEATURES:
---------

	After the jaguar logo is complety display, somes possibilities allows:

If a host computer is detected, a nice ST is displaying at the low right screen position
otherwise a crossed computer will be display.

Press 'A' to start a game.

Press 'B' to reset the jaguar

Press 'C' to send a small string to the host computer
