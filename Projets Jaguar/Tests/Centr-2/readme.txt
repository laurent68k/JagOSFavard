Programme de test avec JToolbox de la communication
ST/jaguar via port Joypad 2/Centronics
===================================================

Celui-ci utilise desormais les routines integrees dans la
boite a outils Jaguar.

repertoires:
------------

	/src	contient les sources du programme TOS .
	/Jaguar	contient le source du programme Jaguar avec
			ses propres routines d'E/S sur Joypad.
			
Celui-ci est aussi un simple terminal. Chaque caractere tape au
clavier est envoye a la jaguar qui le retourne a l'Atari.
